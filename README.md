# Doughlicious-Delights
A project to make a working website for bakery(with all working functions and responsiveness).
First file i.e., entry.html gives us a choices between signup or enter anyway.
From where we goes to main page of our project which shows us all the required content expected from an bakery website.
profile.html - your information , language setting, logout or delete accout fuctions.
All the add to cart and order history,signup data is saved in local device .
This project have a beautiful night mood (through canvas).
Every page is connected to Doughlicious-Delights.css and Doughlicious-Delights.js because of header and footer(except login.html and signup.html).
But cart.html ,order.html ,login.html and signup.html have there own css & javascript (for working).
When products in cart(cart.html) are ordered then we are forwarded to order.html where we watch order detail above past order history.
On signup , we are automatically loged in but in case you logout or something else happen you can login through entry page or profile page .
